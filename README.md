# Minishell

## Description

This project is a limited replica of bash, its goal is
to accurately replicate some of bash functionalities with the 
the following built-in commands:
- `echo with option -n`
- `cd with only a relative or absolute path`
- `pwd with no options`
- `export with no options`
- `unset with no options`
- `env with no options or arguments`
- `exit with no options`

- - - - - - -

## Instructions

### Compilation :
```bash
make 
```

### Parameters : 

- `env -i` : Run Minishell without environment variables

### Usage :

Use like bash
example :
- `ls`
- `echo hello world`
- `pwd`
